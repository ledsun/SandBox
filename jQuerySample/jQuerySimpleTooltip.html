jQuery Simple Tooltip

シンプルに

simpletooltipメソッドでリンクにツールチップを設定

$("a.tooltiplink").simpletooltip();

hrefで表示するdivのIDを指定します

<a href="#ma_tooltip">Lien</a>
<div id="ma_tooltip">Tooltip</div>

表示内容はspanでもOK

デモ

<span id="Span1" class="tooltip">Tooltip 4</span>

ツールチップのスタイルは呼び出し対象のdivに設定

デモ

リンクは#以降しか見ません

デモ

<a id="A1" href="xxx#tooltip_1">デモ</a>

オプション

click

クリックしないと開かない デモ

$("a.clic").simpletooltip({ click: true });

hideDelay

閉じるまで時間がかかる デモ

$("a.delay").simpletooltip({ hideDelay: 0.5 });

showDelay

開くまで時間がかかる デモ

$("a.delay").simpletooltip({ showDelay: 0.5 });

hedeOnLeave

閉じないデモ

$("a").simpletooltip( { hideOnLeave: false } );

ツールチップ内のリンクにrel="close"を指定する。リンクをクリックするとツールチップを閉じます。

<a href="#..." rel="close">閉じる<a>

margin

ウィンドウ端とツールチップの最小幅（デフォルト5）デモ

$("a").simpletooltip( { margin: 10 } );

showEffect et hideEffect

表示と閉じるときのエフェクトを指定できる。エフェクト名はjQuery Effectsのメソッド名と対応しています。

フェードイン、フェードアウト デモ

// Effet "fadeIn"
$("a").simpletooltip( { showEffect: "fadeIn", hideEffect: "fadeOut" } );

スライドイン、スライドアウト デモ

// Effet "slideDown"
$("a").simpletooltip( { showEffect: "slideDown", hideEffect: "slideUp" } );

左上から拡大、左上へ縮小 デモ

// Effet "show"
$("a").simpletooltip( { showEffect: "show", hideEffect: "hide" } );

showCallback

開く前に処理を追加 デモ

$("a").simpletooltip({
	showCallback: function(tooltip){
		window.alert('Callback : affichage de la tooltip #'+tooltip.id);
	}
});

hideCallback

閉じる前に処理を追加 デモ

$("a").simpletooltip({
	hideCallback: function(tooltip){
		window.alert('Callback : affichage de la tooltip #'+tooltip.id);
	}
});

customTooltip

ツールチップをその場で生成 デモ aタグである必要はない。$(target).text()で対象文字列を取得できます。

// Permet de generer dynamiquement une tooltip
$("a").simpletooltip({
	customTooltip: function(target){
		return '<p class="customTooltip">Texte de l\'element survole :<br /> "' + $(target).text() + '" </p>';
	}
});
