4種類のデバイスを想定

  1. スマートフォン
  2. タブレット
  3. 小さいデスクトップ
  4. 大きいデスクトップ
xs(Extra small devices Phones） ~ 768px ~ sm(Small devices Tablets) ~ 992px ~ md(Medium devices Desktops) ~ 1200px ~ lg(Large devices Desktops)

Containerの幅

  1. xs: Auto
  2. sm: 750px
  3. md: 970px
  4. lg: 1170px
1200px以上の画面を最大限に使いたい場合はContainerをそのままでは使えない。
